# instabug-reactnative-ndk

eact Native plugin for integrating the Instabug SDK NDK

## Installation

```sh
npm install instabug-reactnative-ndk
```

## Usage

```js
import { multiply } from 'instabug-reactnative-ndk';

// ...

const result = await multiply(3, 7);
```
